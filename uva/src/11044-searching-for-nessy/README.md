# Searching for nessy

## Solution

```c++
#include <cstdio>

int main()
{
    int t;
    int m, n;

    scanf("%d", &t);

    while(t--) {
        scanf("%d %d", &m, &n);
        printf("%d\n", (m / 3) * (n / 3));
    }

    return 0;
}
```
